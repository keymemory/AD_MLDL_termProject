## A. 파이프라인 구조 대조 (명세 §전체 파이프라인)

명세가 요구한 흐름과 구현 흐름이 1:1 일치하는지:

| 단계 | 명세 요구 | 구현 위치 | 일치 |
|---|---|---|---|
| 입력→576토큰 | CLIP-ViT-L/14(336) → 576 visual tokens | `clip_encoder.py`(VisPruner 원본) | ✅ |
| Stage1 important | [CLS] attn 상위 `M1·r`개 | `llava_arch.encode_images` important_indices | ✅ |
| Stage1 diverse | cosine 중복제거로 `M1·(1-r)`개 | 동 함수 ToMe식 while 루프 | ✅ |
| Stage1 출력 | `M1`개 보존 + **attn score 동반** | `image_attentions.mean(1)` 보유 후 gather | ✅ |
| Stage2 clustering | `M1`→`M2` spherical k-means 병합 | `spherical_kmeans.merge_tokens` 호출 | ✅ |
| 대표토큰 | simple_avg / weighted_avg | `merge_tokens(method=...)` | ✅ |
| LLM 주입 | 최종 `M2`개를 `<image>` 자리 | `prepare_inputs` flatten 주입 | ✅ |
| 분기 | clustering off → 기존 VisPruner 동일 | `enable_clustering`·`M2<M1` 가드 | ✅ |

**파이프라인 결론**: 명세 다이어그램과 구현 데이터 흐름 완전 일치.

## B. Stage 1 명세 (§Stage 1) 항목 대조

| 명세 항목 | 요구 내용 | 구현 | 판정 |
|---|---|---|---|
| 1. [CLS] attn 추출 | select_layer=-2, `attentions[:,:,0,1:]`, head 평균 | VisPruner 원본 그대로(`clip_encoder` + `image_attentions.mean(dim=1)`) | ✅ |
| 2. Important 선택 | attn 내림차순 상위 `T_imp=M1·r` | `argsort(descending) → [:,:important_token_num]` | ✅ |
| 3. Diverse 선택 | 잔여서 cosine 중복 반복 제거, `T_div=M1·(1-r)` | 원본 ToMe식 bipartite while 루프 유지 | ✅ |
| 4. 결합 | important+diverse=M1, 원 패치순서 | `cat` 후 `index_masks.scatter_`(불리언=공간순서 보존) | ✅ |
| 수정 A | M1을 M2보다 크게 설정 가능 | `M1=get_stage1_tokens()`, `visual_token_num=M1`로 선택 | ✅ |
| 수정 B | 1단계서 attn score 함께 반환 | `image_attentions`(B,N) 유지 → 선택 idx로 gather하여 Stage2 전달 | ✅ |

## C. Stage 2 명세 (§Stage 2) 항목 대조

| 명세 항목 | 요구 | 구현 (`spherical_kmeans.py`) | 판정 |
|---|---|---|---|
| L2 정규화 | 단위 구 투영 | `_l2norm` (float32 계산) | ✅ |
| 초기 centroid | 랜덤 k개(또는 attn 상위) | `randperm[:k]` (init_indices 인자도 지원) | ✅ |
| 반복 a | 토큰-centroid cosine | `x @ centroids.t()` | ✅ |
| 반복 b | 최유사 centroid 할당 | `sim.argmax(dim=1)` | ✅ |
| 반복 c | 클러스터 평균→새 centroid | `index_add_` 평균 | ✅ |
| 반복 d | 새 centroid 재정규화 | `_l2norm(...)` | ✅ |
| 반복 e | assignment 불변 시 조기종료 | `torch.equal(assign,prev) → break` | ✅ |
| 대표 (a) simple | 클러스터 단순평균 | `out.index_add_; /cnt` | ✅ |
| 대표 (b) weighted | `Σaᵢxᵢ/Σaᵢ` (attn 가중) | weighted_avg 분기 그대로 | ✅ |
| 주의 GPU 텐서 | numpy 변환 없이 torch | 전부 torch GPU | ✅ |
| 주의 fp16 eps | `eps=1e-8` 0나눗셈 방지 | `_EPS=1e-8` clamp_min | ✅ |
| 주의 max_iter | ≤10 | 기본 10, CLI 조정 | ✅ |
| 주의 빈클러스터 | 최대 클러스터서 1토큰 재할당 | empty 처리 루프 구현 | ✅ |
